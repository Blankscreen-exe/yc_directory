## Installation

1. Use `npm=^10.5.0`, `node=21.7.3`, `next=15.3.0`, `react=^19.0.0` and `react-dom=^19.0.0`

2. Clone the repo and install dependencies

```sh
npm install
```

3. Run the development server

```sh
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
